DO THIS EXACTLY

1. Open your browser.
2. Open this file:
   C:\Dev\Image generator\index.html
3. Set these options:
   - Workflow type = Generate UI and remove the background
   - What kind of game UI do you want? = Pixel Art
   - Which image model are you using? = Juggernaut XL
   - Which background remover do you want? = ComfyUI Essentials / TransparencyBackgroundRemover
   - What are you making? = pixel art inventory icon
   - Extra material or theme details = bronze frame, blue gem center, clean retro game palette
   - I need crisp edges for sprites or pixel-style UI = ON
4. Click:
   Build My Workflow
5. Click:
   Download JSON
6. Save it anywhere, or just use this file I already made:
   C:\Dev\Image generator\starter-workflow-pixel-ui-background-removal.json

COMFYUI INSTALL

1. Open ComfyUI.
2. Open ComfyUI Manager.
3. Install these:
   - ComfyUI Essentials
   - Juggernaut XL checkpoint
4. Restart ComfyUI.

WHAT TO SEARCH FOR IN COMFYUI

Add these nodes in this order:

1. CheckpointLoaderSimple
2. CLIPTextEncode
3. CLIPTextEncode
4. EmptyLatentImage
5. KSampler
6. VAEDecode
7. TransparencyBackgroundRemover
8. SaveImage

HOW TO WIRE THEM

1. CheckpointLoaderSimple:
   - model -> KSampler model
   - clip -> both CLIPTextEncode nodes
   - vae -> VAEDecode vae

2. Positive CLIPTextEncode:
   Paste this:
   pixel art game ui asset, pixel art inventory icon, bronze frame, blue gem center, clean retro game palette, centered, isolated background, limited palette, crisp silhouette, readable shape language, production-ready sprite, sharp edges, game asset sheet, pixel-perfect outline, no fringe, hard alpha edge, Use standard CLIP text encoding with asset-first wording.

3. Negative CLIPTextEncode:
   Paste this:
   blur, anti-aliased smudge, painterly, photographic, text, watermark, extra objects, fuzzy outline, soft lighting, low resolution

4. EmptyLatentImage:
   - width = 1024
   - height = 1024
   - batch size = 1

5. KSampler:
   - seed = 123456789
   - steps = 30
   - cfg = 6.5
   - sampler = dpmpp_2m_sde
   - scheduler = karras
   - denoise = 1
   Also connect:
   - positive CLIPTextEncode -> KSampler positive
   - negative CLIPTextEncode -> KSampler negative
   - EmptyLatentImage -> KSampler latent_image

6. VAEDecode:
   - KSampler samples -> VAEDecode samples

7. TransparencyBackgroundRemover:
   - VAEDecode image -> TransparencyBackgroundRemover image
   - if you see edge mode, set it to PIXEL_ART

8. SaveImage:
   - TransparencyBackgroundRemover output -> SaveImage images
   - filename prefix = pixel_ui_asset_transparent

FINAL STEP

1. Click Queue Prompt.
2. Wait for the image to generate.
3. Look in your ComfyUI output folder.
4. The saved image should be a PNG with transparency.

IF YOU GET STUCK

If ComfyUI says a node is missing:
- You did not install ComfyUI Essentials
- or the node pack needs a restart

If Juggernaut XL is missing:
- put the model file in your ComfyUI checkpoints folder
- restart ComfyUI
