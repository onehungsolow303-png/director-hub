AI Mode: Enhanced Pipeline Architecture

ComfyUI segmentation + edge protection + spill cleanup

Pipeline Flow
STEP 1: ComfyUI Segmentation
Source image → ComfyUI proxy → BiRefNet/RMBG/etc → Raw AI mask
Uses whichever model is selected in the Segmentation Model dropdown
↓
STEP 2: Edge Protection (NEW)
Detect high-contrast edges in original image (Sobel-like gradient)
Where AI mask says "remove" but edge strength is high → bias toward keep
Protects ornate borders, gold scrollwork, stone texture edges from being eaten by AI
↓
STEP 3: Spill Cleanup (ENHANCED)
For edge pixels (partial alpha): remove color contamination from game world
Existing unpremultiply + channel-limit clamping → clean transparent edges
Uses the existing decontamination math, applied specifically to AI mask boundaries
↓
STEP 4: Final Compositing
Refined alpha + source image → transparent PNG with UI only
Split panel detection, manual corrections, download — all existing tools still work

What Changes

Current AI Mode
  • AI mask comes back from ComfyUI
  • Applied directly as alpha
  • Basic decontamination on edges
  • Ornate details often lost
Enhanced AI Mode
  • AI mask comes back from ComfyUI
  • Edge detection protects UI details
  • Enhanced spill cleanup at boundaries
  • Ornate borders and textures preserved

No UI Changes Needed

The refinement runs automatically inside the existing AI mode pipeline. No new buttons, toggles, or modes. User clicks "Generate AI Mask" → "Process Image" same as before — just gets better results on complex scenes.
